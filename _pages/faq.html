What are the system requirements?

Windows Vista, 7, 8.1, and 10 are supported. Windows XP was once supported, but is no longer supported because most Windows XP computers are compatible with the latest Linux distros, which contain Wine.

Linux (native 64-bit only):

  • Linux Mint 18
  • Ubuntu 16.04 LTS (Xenial)
  • Arch Linux
  • Debian 9 (stretch)
  • Gentoo 20160514
  • Fedora 24
  • openSUSE 42.2
  • …and other Linux distros with a Qt version ≥5.5.1.

Regardless of your operating system, you will also need at least 512 MB RAM, 2 GB of storage, stable Internet access, and a brain.

Can I play offline cases?

Unfortunately, the design of the game does not allow for offline case support to be added. We would like to incorporate this functionality into future versions of Attorney Online, however.

Can I play this on my phone?

We're working on it! Check out our web client, which is currently under development.
